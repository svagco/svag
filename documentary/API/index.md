
## API

The package is available by importing its named exports.

```js
import {
  Window,
  Shadow,
  Toolbar,
  lib,
} from 'svag'
```

```## Window
[
  ["options", "WindowOptions"]
]
```

Creates a [`@svag/window`](https://github.com/svagco/window).
